# Web-приложение «Интерактивный анализ конкурентного рынка»

## Актуальность
Создание данного web-приложения важно по нескольким причинам:

Во-первых, всё чаще в современном образовании применяются цифровые технологии, которые позволяют успешнее осваивать материалы курсов по экономике.  
С этой точки зрения эффективно использование такого инструмента, как web-приложение «Интерактивный анализ конкурентного рынка», поскольку оно будет не только помогать с решением задачи, но и объяснением шагов и их визуализацией.  

Во-вторых, представляется интересным совмещение прикладной экономики и языка программирования Python воедино.  
Это позволит практиковаться не только в решении экономических задач, но и в отработке навыков программирования, поскольку в силу современных тенденций связь экономики и программирования важна для востребованности на рынке труда.

## Описание
Данное web-приложение позволит выполнять следующие задачи:
- Расчёт равновесия на совершенно конкурентном рынке при различных условиях (налоги, субсидии);
- Визуализация поэтапного решения задачи с дополнительной теорией;
- Наблюдение за динамическим изменением рынка посредством изменения параметров спроса и предложения, а также налогов и субсидий;
- Решение задач на схожие темы во встроенном редакторе Python с автопроверкой корректности решения.

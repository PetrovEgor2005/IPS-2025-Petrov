# Web-приложение «Интерактивный анализ конкурентного рынка»

## Актуальность

На мой взгляд, создание данного web-приложения важно по нескольким причинам: во-первых, всё чаще в современном образовании применяются цифровые технологии, которые позволяют успешнее осваивать материалы курсов по экономике. С этой точки зрения эффективно использование такого инструмента, как web-приложение «Интерактивный анализ конкурентного рынка», поскольку оно будет не только помогать с решением задачи, но и объяснением шагов и их визуализацией. Во-вторых, представляется интересным совмещение прикладной экономики и языка программирования Python воедино, это позволит практиковаться не только в решении экономических задач руками, но и отработке навыков программирования, поскольку в силу современных тенденций связь экономики и программирования важна для востребованности на рынке труда.

## Описание

Приложение будет выполнять следующие задачи:

### 1. Расчёт равновесия на конкурентном рынке
- Рассчёт равновесия при различных условиях (налоги, субсидии).
- Поддержка линейных и нелинейных функций спроса и предложения.

### 2. Интерактивная визуализация
- Построение графиков спроса/предложения, отображение равновесия и сдвигов кривых.
- Визуализация влияния налогов и субсидий на рынок.
- Слайдеры для параметров, мгновенное обновление графика.

### 3. Сценарии и сохранение результатов
- Возможность сохранять сценарии (наборы параметров) для дальнейшего использования.
- Сравнение 2–3 сохранённых сценариев на одном графике.
- Хранение сценариев в базе данных (SQLite/PostgreSQL).

### 4. Генератор задач
- Автоматическая генерация задач с параметрами, выбранными случайным образом в разумных диапазонах.
- Автоматическое вычисление верного решения задачи.
- Возможность решения задачи как вручную, так и через встроенный Python-редактор с автопроверкой решения.

### 5. Встроенный редактор Python
- Студент может писать решение задачи в редакторе Python.
- Проверка правильности решения с использованием юнит-тестов.
